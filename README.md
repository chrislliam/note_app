# note_app
 App Note To Do List
 
# Features
 * To do list usando sqlite como db
 * Página edit para criar e editar nota
 * Função para mudar tema entre light e dark
 
# Dev Tools
 App feito utilizando apenas ChangeNotifier e Provider
